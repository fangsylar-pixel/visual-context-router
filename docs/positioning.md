# Positioning

Visual Context Router is not a screenshot compressor. It is a routing layer for visual context.

## One-Line Pitch

Stop sending full screenshots to agents.

## Slightly Longer Pitch

Visual Context Router lets Codex and other AI agents inspect screens through local change detection, wireframe prompts, and ROI crops before spending tokens on full screenshots.

## What It Is

- A local Python CLI and SDK.
- A Codex plugin with MCP tools.
- A screen observation router for agent workflows.
- A practical experiment in reducing repeated visual context cost.

## What It Is Not

- It is not a global Codex screenshot interceptor.
- It does not guarantee token savings unless the agent calls its tools.
- It is not a replacement for full screenshots when global visual context matters.
- It is not a mature computer vision UI parser yet.

## Best Use Cases

- Repeated screen observations in Codex.
- Browser and desktop automation agents.
- RPA-style workflows.
- UI testing and debugging loops.
- Token savings experiments for multimodal agents.

## Suggested Tagline

```text
A token throttle for screen-reading AI agents.
```
